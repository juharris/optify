# Optify Rust Bindings for Ruby

## Setup
<!-- Some tips in https://github.com/matsadler/magnus/issues/77 -->

Use Ruby 3.3+., for example, run:
```shell
chruby 3.3.1
```

Run:
```shell
bundle install
```

## Building
Run:
```shell
rake compile
```
